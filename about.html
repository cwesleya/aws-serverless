About Me

My name is Wesley Chambers and this is my personal site. I'm not sure where I want to go with it yet, but I've written it using only HTML, CSS, and plain Javascript. The actual CI/CD process is automated and uses mostly AWS services. 

Right now it is a static site registered in Route53 and hosted in S3 behind a CloudFront distribution. When code is pushed to the master branch in GitHub, a CodePipeline process kicks off. It takes the code from this repository, makes a build using CodeBuild (based on the buildspec.yml), which zips the built code and puts it into a separate S3 bucket. The pipeline then executes a Lambda function, written in python, which unzips and publishes the site to the site bucket. The function then uses SNS to send an email to the admin (me!).
